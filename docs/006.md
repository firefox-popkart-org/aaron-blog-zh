# 

> 原文：<http://www.aaronsw.com/weblog/001112>

*《纽约时报》前期版让我为他们正在下载的观点/对位法贡献一篇短文。(当然，我会为下载辩护。)我想我设法在几天内写了一篇相当好的文章，尤其是对它的规模和观众来说。但是后来我发现我的作品被剪了，因为《泰晤士报》决定不告诉孩子们违法。所以，从墓地，它在这里。*

偷窃是错误的。但是下载不是偷窃。如果我在当地唱片店顺手牵羊买了一张专辑，其他人都买不到。但是当我下载一首歌的时候，没有人会失去它，而另一个人会得到它。不存在道德问题。

音乐公司将 2000 年以来 15%的销量下降归咎于下载。但在同一时期，出现了经济衰退、价格上涨、新专辑降价 25%，以及缺乏受欢迎的新艺人。考虑到所有这些，也许下载*可以增加*的销量。90%的主要标签目录不再出售。互联网是听到这种音乐的唯一途径。

即使下载确实损害了销售，但这并不意味着它不道德。图书馆和音像店(两者都不按租金付费)也损害了销售。使用它们是否不道德？

下载可能是非法的。但是有 6000 万人使用 Napster，只有 5000 万人投票给布什或戈尔。我们生活在一个民主国家。如果人们想分享文件，那么法律应该改变，让他们。

有一个公平的方法可以改变它。一位哈佛教授发现 60 美元/年。对宽带用户收费将弥补所有收入损失。政府将把它给受影响的艺术家，作为回报，使下载合法，激发更容易使用的系统和更多的共享音乐。艺术家得到更多的钱，你得到更多的音乐。这有什么不道德的？

脚注:

“自 2000 年以来销售额下降了 15%”:这来自 RIAA 自己的图表。1999 年他们卖出了 9.389 亿张 CD，2002 年他们卖出了 8.033 亿张(938.9-803.3)/938.9 ~= .14(所以真的更接近 14%，不过我们暂且不说，先说 15%)。

“新版本降价 25%”:这取决于你如何计算。RIAA 说，他们在 1999 年发行了 38900 部新电影。根据声音扫描，RIAA 在 2001 年发布了 31734 个新版本，导致了 18%的下降。这真的不公平，因为我们使用的是 1999 年的 RIAA 数字和 2001 年的 SoundScan 数字，而且 SoundScan 可能没有 RIAA 那么多专辑。然而，RIAA 在 2003 年初表示，他们在前一年发行了 27，000 张新专辑。显然被这些信息弄得很尴尬，他们已经从网站上删除了这些信息。但是如果你用他们的数字，你会得到 31%的下降。我把差额分成两部分，称之为 25%的削减。但是如果你愿意，我可以把这个变成 30%或 20%；我认为这不会改变争论。

引人入胜的辩论和思想交流。不幸的是，似乎有许多概念的融合，可能更好地分开考虑。如果可以的话:

1)造物主的控制(或“权利”):我们当前的“知识产权”(我讨厌这个词的含义)制度的(简短)历史和先例可以追溯到一切都属于君主的时候，从一个人的努力中受益的特权是由君主单独授予的。这在普通法中变成了一种有限期限和范围的垄断，之后所有作品的所有权都回归到现在的公共领域。我们错综复杂的商标、专利和版权(在大多数西方国家)试图平衡公共和私人利益，同时鼓励艺术和科学的持续发展。重点是“平衡”这个词。创作者的权利必须与进步相平衡，当所说的“知识产权”主要被用来扼杀创新和进步(如“防御性专利组合”的集合)或有效维持契约奴役(“感谢录制合同，RIAA！”)而不是公平地补偿艺术家或创作者。

受《伯尔尼版权公约》等所有平衡制约的创造性控制，在不断发展的知识共享许可制度、拟议的《埃尔德里德法案》以及迫切需要的专利审查过程中，比现在不合时宜的现行过程更好地实施。

2:对艺术家的补偿:是的，艺术家应该得到补偿，而且，众所周知，许多与 RIAA 会员公司签约的音乐人基本上得不到公平的补偿。无线电模式运行良好，政府征收“税收资助创意”的提议——无论如何实施(例如，通过对加拿大目前存在的可记录媒体征收关税、对宽带互联网连接征收关税等)。)-在一个开明、文明的社会中是一种社会公益。“我不下载的论点；“我不应该补偿那些这样做的人”只是一种卑鄙的想法，与类似的观点如道路养护、教育税和类似的社会公益筹资机制相呼应。正如其他地方(尤其是独立音乐人)所证明的那样，他们倾向于通过可下载的音乐(比如他们的“后台列表”)来制作更多音乐，以推广他们的新音乐和现场音乐会。这在一定程度上恢复了 20 世纪 50 年代的广播，尽管有贿赂。

3:合理使用和优先销售权:这些是版权法妥协试图平衡的社会契约的一部分。这些都是公认的原则，很少有人会(或者能够)合理地反对。我们现在面临的问题是，合同法正被用来规避和违反社会契约，但更重要的是，版权法，它已经把 FU(没有双关语)和 RFS 作为法律。通过强制实施数字版权管理机制，用户可以“被迫”接受限制或完全取消其 FU 和 RFS 权限的 EULA 协议，也就是说，如果他想使用有问题的软件。如果该软件是无处不在的微软 Word 或 Windows Media Player，并且其中的许多内容都是独家存在的，那么用户显然不必同意 EULA 的选择就会受到严重限制。在我们应用可信计算和数字千年版权法之前，这是一个很好的论据，可以替代像 Word 和 WMP 这样的内容管理应用程序。开源替代方案不会被“信任”，任何绕过信任的尝试从表面上看都会违反 DMCA。对各种各样的既得利益公司来说，这是一个大灌篮，对公众来说，这是创新能力的巨大损失——劳伦斯·莱斯格称之为“修补的权利”。

从历史上看，那些在历史上任何时候都处于领先地位的人都希望保持领先地位，并尽最大努力防止任何颠覆性创新。我们已经多次看到这种情况，也许最生动的例子是著名的索尼 VCR 诉讼。电视节目的家庭录制被认为是盗版，就像今天下载音乐被认为是盗版一样，有着同样的可怕后果:媒体的死亡，没有制作新作品的动机，艺术家得不到补偿，等等。我们都知道发生了什么。家庭中录像机的激增导致了录像带电影的增长，在过去的十五年里，大多数电影公司都从录像带中获得了全部利润。当然，电视会扼杀电影，互联网会扼杀出版业…

作为一个社会，我们非常不擅长线性预测，尤其是那些行业预测世界末日情景的人。在我看来，问题不在于下载是否是偷窃，甚至不在于是否道德。问题是，在我们的社会/商业/工业/诸如此类的动态中，有哪些看不见的结构性变化，从中可以创造新的商业机会。从我们在历史上一次又一次看到的情况来看，作为一个新业务发展起来的东西，其利润是我们希望保持的旧业务的很多倍。

现在，回到手头的问题:下载是偷窃吗？亚伦，你是正确的，一份数字作品的拷贝并没有剥夺我自己的作品，但它*确实*剥夺了我失去的收入机会(无论多么假设)。对宽带征收 60 美元/年的“税”并不是一个答案——就我而言，我不下载音乐，因此这将是对我不希望的东西的 60 美元/年的浪费。即便如此，人们如何确定谁会得到补偿？像 ASCAP、身体质量指数或 SESEC 使用的统计方法？严格的记录保存？谁保存这些记录？它们是如何编制的？保存记录是*昂贵的*；你意识到仅仅保留*电话记录*有多大问题吗？现代电话交换机的大部分复杂性来自于维护准确的电话使用记录。
